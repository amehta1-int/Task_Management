# Task Manager

A simple Rails application for managing tasks with user authentication.

## Features

- User registration and authentication
- Create, edit, and delete tasks
- Task status tracking (Todo, In Progress, Done)
- Task priority levels (Low, Medium, High)

## Requirements

- Ruby 3.2.6
- Rails 8.1.2
- MySQL 5.7+

## Setup

```bash
# Install dependencies
bundle install

# Setup database
rails db:create db:migrate

# Start server
rails server
```
